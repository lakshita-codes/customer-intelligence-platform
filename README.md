# AI-Powered Customer Intelligence & Churn Analytics Platform

An end-to-end business analytics and AI platform for customer churn prediction, sentiment analysis, KPI tracking, and AI-generated business insights.

## Tech Stack
- Python
- SQL
- PostgreSQL
- FastAPI
- Streamlit
- Power BI / Tableau
- Machine Learning
- NLP & LLM APIs
- AWS

## Features
- Customer churn prediction
- Sentiment analysis
- Complaint categorization
- KPI dashboards
- REST APIs
- AI-generated business recommendations
